Welcome to Jutellane Docs

This site is the single source of truth for how we ship at Jutellane—our standards for DevSecOps, cloud reliability, and sustainable engineering. You’ll find concise, practical guidance that mirrors the way we actually work in projects.

What you’ll find

  • Getting Started — how to browse, run locally, and contribute.
  • Tooling Setup — linters, formatters, precommit, IDE hygiene.
  • CI/CD Pipelines — PR previews, production static deploy, quality gates.
  • Architecture — high-level diagrams for services and data flows.

How to use these docs

  1. Start with Getting Started for ground rules.
  2. Match your editor to Tooling Setup to catch errors early.
  3. Follow CI/CD Pipelines when opening PRs or shipping to prod.
  4. Skim Architecture when orienting across systems.
